# FastAPI REST API with PostgreSQL

This project is a **backend REST API built with FastAPI and PostgreSQL**.  
It demonstrates how to design a structured backend service with database integration using Python.

The project focuses on building a **clean API architecture**, handling HTTP requests, and storing data in a relational database.

Repository: https://github.com/emrepeker/api_project

---

## Features

- REST API built with **FastAPI**
- PostgreSQL database integration
- JSON request and response handling
- Modular API structure
- Automatic API documentation with **Swagger / OpenAPI**

---

## Technologies Used

- Python
- FastAPI
- PostgreSQL
- REST API
- JSON
- Git
